# CDGP数据治理专家考试学习计划

## 考试信息
- **考试名称**: CDGP (Certified Data Governance Professional) 数据治理专家
- **考试日期**: 2026年3月22日
- **备考时间**: 3周（2026年3月1日 - 2026年3月21日）

---

## 学习阶段规划

### 第一阶段：基础理论（3月1日 - 3月7日）

| 日期 | 学习内容 | 学习目标 |
|------|----------|----------|
| 3月1日 | 数据治理概述、DAMA-DMBOK框架 | 理解数据治理定义、范围、原则 |
| 3月2日 | 数据治理组织架构与角色职责 | 掌握数据治理组织设计方法 |
| 3月3日 | 数据标准管理（词根、数据元、数据字典） | 掌握数据标准制定流程 |
| 3月4日 | 数据标准管理实践案例 | 能够进行数据标准落地分析 |
| 3月5日 | 元数据管理基础 | 理解元数据分类、元数据管理流程 |
| 3月6日 | 元数据管理工具与方法 | 掌握元数据采集、存储、应用 |
| 3月7日 | 第一阶段复习与练习题 | 巩固知识点，完成练习 |

### 第二阶段：核心领域（3月8日 - 3月14日）

| 日期 | 学习内容 | 学习目标 |
|------|----------|----------|
| 3月8日 | 数据质量管理概述 | 理解数据质量维度、评估方法 |
| 3月9日 | 数据质量规则与检核 | 掌握质量规则设计、问题分析 |
| 3月10日 | 数据安全管理基础 | 理解数据安全分级、脱敏技术 |
| 3月11日 | 数据安全合规与隐私保护 | 掌握数据安全治理实践 |
| 3月12日 | 主数据管理概述 | 理解主数据识别、编码规则 |
| 3月13日 | 主数据管理实践 | 掌握主数据管理流程与工具 |
| 3月14日 | 第二阶段复习与练习题 | 巩固知识点，完成练习 |

### 第三阶段：进阶与实践（3月15日 - 3月21日）

| 日期 | 学习内容 | 学习目标 |
|------|----------|----------|
| 3月15日 | 数据架构管理 | 理解企业数据架构设计 |
| 3月16日 | 数据生命周期管理 | 掌握数据全生命周期管理方法 |
| 3月17日 | 数据资产估值与运营 | 理解数据资产化、价值评估 |
| 3月18日 | 数据治理平台与工具 | 了解主流数据治理工具 |
| 3月19日 | 综合案例分析 | 掌握数据治理项目实施方法 |
| 3月20日 | 模拟考试与错题复习 | 检验学习成果，查漏补缺 |
| 3月21日 | 重点知识回顾与考前准备 | 强化记忆，调整状态 |

## 知识体系大纲

### 1. 数据治理基础
- [ ] 数据治理定义与价值
- [ ] DAMA-DMBOK知识体系
- [ ] 数据治理成熟度模型
- [ ] 数据治理组织架构

### 2. 数据标准管理
- [ ] 业务术语标准
- [ ] 数据元标准
- [ ] 数据字典管理
- [ ] 标准落地与执行

### 3. 元数据管理
- [ ] 技术元数据
- [ ] 业务元数据
- [ ] 操作元数据
- [ ] 元数据血缘分析

### 4. 数据质量管理
- [ ] 数据质量维度（完整性、准确性、一致性、及时性等）
- [ ] 数据质量规则设计
- [ ] 数据质量检核与问题处理
- [ ] 数据质量报告与改进

### 5. 数据安全管理
- [ ] 数据安全分级分类
- [ ] 数据脱敏技术
- [ ] 数据访问控制
- [ ] 数据安全合规（个人信息保护法、数据安全法）

### 6. 主数据管理
- [ ] 主数据识别与定义
- [ ] 主数据编码规则
- [ ] 主数据管理流程
- [ ] 主数据分发与同步

### 7. 数据架构管理
- [ ] 企业数据架构
- [ ] 数据模型设计
- [ ] 数据仓库架构
- [ ] 数据集成架构

### 8. 数据生命周期管理
- [ ] 数据创建与采集
- [ ] 数据存储与维护
- [ ] 数据归档与销毁
- [ ] 数据保留策略
